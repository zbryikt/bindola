What is Bindo.la?

bindo.la! 翻桌啦 is an experimental online voting service ( tentative ) that helps you decide things. It originates from
g0v

Everything here is in progress and is subject to change, but this site will mainly focus on the "voting" topic.




Schulze.js

JavaScript implementation of Condorcet election method using Schulze Algorithm. For using along with Node.js or directly in browsers.

Condorcet GSheet Add-on

The Condorcet for Google Spreadsheet Add-on that helps you compute ranking from scores of candidates with Schulze Algorithm. Easy to use as a simple function call.
